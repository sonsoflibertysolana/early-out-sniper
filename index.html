DESKTOP (EASIEST)

  1. Enter your weekly shifts in the Schedule tab
  2. Go to the Sniper tab when it's close to gate time
  3. Click COPY SNIPER SCRIPT
  4. Open Casino Schedule Ease in another tab
  5. Navigate to My Schedule and click your shift so "Early Out" is visible
  6. Press F12 > Console tab > type allow pasting > Enter
  7. Paste the script > Enter
  8. The sniper arms automatically and fires at the right time

IPHONE (SAFARI)

  1. Add this page to your Home Screen (Share > Add to Home Screen)
  2. Enter your schedule
  3. When the countdown is close, open Safari
  4. Go to Casino Schedule Ease > My Schedule > click your shift
  5. Go to the Sniper tab and tap COPY SNIPER SCRIPT
  6. In Safari's address bar, type javascript: then paste the script
  7. Alternative: Create a bookmarklet:
    1. Bookmark any page in Safari
    2. Edit the bookmark
    3. Replace the URL with the script from COPY SNIPER SCRIPT
    4. When on the CSE page, tap that bookmark to fire

ANDROID (CHROME / KIWI)

  1. Add this page to your Home Screen (Menu > Add to Home screen)
  2. Enter your schedule
  3. Kiwi Browser (recommended): supports Tampermonkey extension for automatic execution
  4. Chrome: Use the bookmarklet method:
    1. Tap COPY SNIPER SCRIPT
    2. On CSE page, tap the address bar
    3. Type javascript: then paste
    4. Chrome may strip javascript: prefix — you'll need to retype it
  5. Firefox: Address bar method works. Type javascript: then paste.

SHARING WITH FRIENDS

  1. Set up your schedule
  2. Go to Schedule tab > COPY SHARE LINK
  3. Send the link to your friend
  4. They open it and the schedule is pre-loaded
  5. They still need their own login credentials from the admin

TIPS

  1. The script auto-detects the correct fire time based on your schedule
  2. If you're already past the gate time, it fires immediately
  3. 200ms buffer is built in so the server has time to open the gate
  4. If "not yet available" appears, the script auto-retries (close > reopen loop)
  5. Max 100 retries (~30 seconds) before it gives up
  6. Make sure your shift is clicked/expanded so "Early Out" link is visible BEFORE the script runs
