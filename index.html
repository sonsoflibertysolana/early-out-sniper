DESKTOP

  1. Enter your shifts in the Schedule tab
  2. Go to Sniper tab near gate time
  3. Click COPY SNIPER SCRIPT
  4. Open the scheduling portal in another tab
  5. Navigate to My Schedule and click your shift so "Early Out" is visible
  6. Press F12 → Console → type allow pasting → Enter
  7. Paste the script → Enter
  8. Click "Early Out" to let the script capture your shift data
  9. Leave it — script fires the API directly at gate time

iPHONE (SAFARI)

  1. Add this page to Home Screen
  2. Enter your schedule
  3. Open Safari → go to portal → My Schedule → click shift
  4. Tap COPY SNIPER SCRIPT
  5. In Safari address bar, type javascript: then paste
